# Temperature and Student Data Analysis

This project is a simple Python assignment that analyzes temperature data and student marks.

In the first task, NumPy arrays are used to analyze temperature data of three cities. The program finds the maximum temperature in City A, determines the day with the highest temperature across all cities, and compares which city is generally warmer.

In the second task, a dictionary of student marks is analyzed. The program finds maximum and minimum marks for each subject, identifies the highest and lowest scoring subjects for each student, and ranks students based on their total marks.

Bar chart visualizations are also included to present the results in a clear and easy-to-understand way.

Tools used: Python, NumPy, Pandas, and Matplotlib.
